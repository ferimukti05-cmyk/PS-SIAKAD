# PS-SIAKAD
Aplikasi Sistem Informasi Akademik (SIAKAD) dirancang untuk mengelola data akademik seperti mahasiswa, dosen, mata kuliah, jadwal, dan nilai secara terintegrasi. Aplikasi ini bertujuan meningkatkan efisiensi, akurasi, dan kemudahan akses informasi akademik bagi seluruh pengguna.https://drive.google.com/file/d/1ApXGcUJg5wnLq_ihg_LHpfBzc1smJGL3/view?usp=drive_link
